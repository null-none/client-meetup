
The Meetup API provides simple client for exploring and interacting Meetup platform from your own apps.


=======
Install
=======

.. code-block:: bash

    pip install client_meeetup

=======
Example
=======

.. code-block:: python

    from meeetup.client import MeetUpScraper

    API_KEY = 'API_KEY'
    MeetUpScraper(API_KEY, 'get_event')
